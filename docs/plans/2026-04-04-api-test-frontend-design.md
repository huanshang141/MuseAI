# API 测试前端页面设计

## 概述

创建一个 Vue 3 + Vite 构建的前端页面，用于开发和调试 MuseAI 后端 API。

## 目标

- 提供直观的界面测试所有现有 API 端点
- 显示请求和响应数据
- 支持开发调试场景

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **样式**: 内联样式或 CSS (保持简单)
- **HTTP 客户端**: fetch API (无需额外依赖)

## API 端点覆盖

### 健康检查模块
- `GET /api/v1/health` - 基础健康检查
- `GET /api/v1/ready` - 就绪检查（含数据库状态）

### 文档管理模块
- `POST /api/v1/documents/upload` - 上传文档
- `GET /api/v1/documents` - 列出所有文档
- `GET /api/v1/documents/{id}` - 获取单个文档
- `GET /api/v1/documents/{id}/status` - 获取文档处理状态
- `DELETE /api/v1/documents/{id}` - 删除文档

## 数据流

1. 用户点击按钮触发 API 调用
2. 显示加载状态
3. 接收响应并显示 JSON 结果
4. 错误时显示错误信息
